SOQL List Formatter

Formatta facilmente le tue liste per query SOQL

Come usare SOQL List Formatter

  • Cosa fa: Converte un elenco di valori in un formato compatibile con le query SOQL di Salesforce.
  • Quando usarlo: Perfetto quando devi filtrare record in Salesforce usando la clausola IN.
  • Come usarlo:
    1. Incolla la tua lista di valori (uno per riga)
    2. Clicca 'Formatta' o usa Ctrl+Enter
    3. Copia il risultato (Ctrl+C)
